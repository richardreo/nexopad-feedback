# 🚀 NexoPad - Public Roadmap & Issue Tracker

Welcome to the official feedback repository for **[NexoPad](https://nexopad.app)**.

NexoPad is the ultimate all-in-one browser workspace designed to kill context-switching. It seamlessly combines a blazing-fast Text Expander, Inline BYOK AI, encrypted Markdown notes, and an advanced link manager.

## 🌟 Core Features

### ⚡ Text Expander & Interactive Forms
* **Dynamic Forms:** Use `{{input}}` or `{{dropdown}}` to trigger mini-forms before sending a message.
* **Live Context Capture:** Magically extract variables like `{{name}}` or `{{company}}` from the active webpage (e.g., a LinkedIn profile).
* **Spintax & Rich Editor:** Rotate text automatically to sound natural and use full HTML/Markdown.

### 🤖 Inline AI Assistant (BYOK)
Connect your own API keys at cost without expensive monthly wrappers: **OpenAI, Anthropic Claude, Google Gemini, DeepSeek, xAI Grok, Groq, or Mistral**.
* **Magic Floating Bubble:** Select text anywhere on the web to Translate, Summarize, or Fix Grammar inline.
* **Direct Injection:** Replace original text with AI output in one click, preserving format.

### 🔒 Secure Notes & Web Clipper
* **Markdown Web Clipper:** Highlight text and press `Alt+C` to save it locally with the source URL.
* **Floating Post-its & PIN Encryption:** Keep your data secure and always on top.

### 🔗 Link & Tab Manager
* **Forced Tab Renaming:** Automatically rename duplicate CRM/Support tabs to identify them instantly.
* **Spotlight Injector (//):** Type `//` in any text field to search and inject snippets without touching the mouse.
